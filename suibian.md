# Smart Wardrobe

# 功能简介
“电子衣橱” (Digital Wardrobe) 或 “智能穿搭助手”。

实现的功能：
- 导入衣物：
  - 根据淘宝，PDD，京东购物记录api导入购入衣物的信息；
  - AI识别抠图，导入衣物
- 穿搭推荐：
  - stable diffusion生成试穿数字人，生成穿上指定衣服的形象
  - 调用当地天气api，推荐着装，下雨暴晒，旅行携带的衣物推荐
  - 互联网分析，得到热门穿搭，给出推荐（AI跨平台分析）；同时也可推荐购置的衣服


# 痛点定位 (Pain Point Localization)
1. 核心需求点：
*   **“衣服多但没得穿” (Wardrobe Paralysis)：** 很多人衣服塞满柜子，却只穿那20%。用户忘记了自己有什么。
*   **搭配困难：** 不知道这件新买的衬衫能配哪条旧裤子。
*   **强迫症与整理收纳：** 喜欢数字化管理生活的人群（Quantified Self）。
*   **可持续时尚：** 现在的趋势是“胶囊衣橱”（Capsule Wardrobe），鼓励少买精买，多重复利用

2. 市场痛点（机会点）：
*   **天气关联：** 确实是刚需。例如：“明天降温且下雨，请提醒我带伞并穿上那件风衣。”
*   **购买决策辅助：** 在买新衣服时，如果有App告诉用户“你已经有3件类似的了”或者“这件衣服能和你衣柜里的5件衣服搭配”，转化率或理性消费感会提升。
 
 总的来说：解决懒人穿衣、买衣的问题

# 技术分析（为何现有竞品不够好）：

局限性： 指出目前的竞品（如Acloset等）最大的技术瓶颈是“录入成本高”（需要手动拍照抠图）和“推荐生硬”（简单的颜色匹配，不懂风格）。现有的AI多用于简单的图像识别，缺乏生成式能力。
